Uses

Make sure to check out uses.tech for a list of everyone's /uses pages!

I change aspects of my setup fairly often so this page will serve as a living document and a place to point curious developers to when I get asked.

Editor + terminal

  • My current editor is (Neo)Vim. This is my current vimrc and here is my current plugins-config.
    About my theme, I switch frequently from gruvbox and one.vim.
  • My development browser is the beloved Firefox Dev Edition and my current personal browser is Brave.
    For my other work that doesn't concern dev, I use Chrome.
  • For my terminal, I'm currently using Windows Terminal with Ubuntu 20.04 LTS. I use Mintty before but when windows terminal got stable, I switched right away. I'm still waiting for Italics tho.
  • If you are interested on my other configs (zsh, bash, git-alias), you can check my dotfile repo. Please keep in mind that although they work for me, they might not be the best for you.

Apps

  • I'm a huge fan of Wox. But when Windows Powertoys Run got a little bit stable, I switched right away. It is slower than Wox but it gets the job done. I still miss my Wox sometimes.
  • When I search for files I use Everything. It is really really fast. It's like magic.
  • I design at Adobe XD. Because well... they have free version and it works.
  • I use Autohotkey to remap my keyboard to my taste. Although Powertoys got Keyboard Manager it sometimes doesn't work for me, so I still haven't switched.
  • The color picker I use before is the one that is built in on Firefox Dev. But these days I use Powertoys color picker, because it is faster and is easier to access.
  • I highly suggest Windows Powertoys. If you haven't use it, you are a Mad Lad!

Desk setup

  • My laptop is Acer Aspire E15. I upgraded the RAM to 12GB and fucked the keyboard so it doesn't work now. I don't suggest this laptop because you can't fix it easily when something is wrong. I'm planning to buy another laptop next year.
  • The keyboard I currently use is Rakk Kali a low profile mechanical keyboard.
  • I am sitting at my Mesh Chair. These days I don't use it anymore because I am practicing to sitting at floor while working.
  • My desk is Rakk Malyari. But when I am working at floor I use the little desk that I bought at local store.
  • If you are working from home, I suggest to work at floor, at first it is hard but it will help your body and posture in long run. Your body is your main arsenal. Invest for it.

Other Gears

  • I'm a simple man, I use Hario Coffee Mill Ceramic Slim to grind my coffee.
  • My french press is GreenBank French Press 600 ml that I bought at local store.
  • I use Hario V60 Drip Coffee Scale to measure my coffee and water volume.
