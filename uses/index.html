Uses

Make sure to check out uses.tech for a list of everyone's /uses pages!

I change specs of my setup fairly often so this page will serve as a living document and a place to point curious developers to when I get asked.

Editor + terminal

  • My current text editor is (Neo)Vim. This is my current vimrc and here is my current plugins-config.
  • About my theme, I switch frequently from Gruvbox and One Vim. But my current favorite theme is Catppuccin.
  • I used DroidSansMono Nerd Font Mono font for years, but this 2022 I switched to Iosevka font.
  • My development browser is the beloved Firefox Dev Edition and my current personal browser is Brave.
    For my other work that doesn't concern dev, I use Chrome.
  • For my terminal, I'm currently using Wezterm with WSL Ubuntu 20.04 LTS inside a nested TMUX. I used Mintty and then Windows Terminal but Wezterm has a more minimalistic look. If you are curious, here is my Wezterm config.
  • If you are interested in my other configs such as zshrc, bashrc, gitconfig, you can check my dotfile repo. Please keep in mind that my configs are personal and opinionated, although they work for me, they might not be the best for you.

Apps

  • I'm a huge fan of Wox. But when Windows Powertoys Run got a little bit stable, I switched right away. It is still slower than Wox but it gets the job done.
  • When I search for files I use Everything. It is really really fast. It's like magic.
  • I design website templates and prototypes in Adobe XD.
  • I use Autohotkey to remap keyboard, macro and autocompletions (like sql queries). Although Windows Powertoys got Keyboard Manager, it doesn't work like autohotkey, so I still haven't switched.
  • The color picker I use before is the Firefox Dev Eyedropper. But these days I use Powertoys Color Picker because it is faster and is easier to access.
  • If you are a Windows User, I highly suggest Windows Powertoys. It will be a great addition to your workflow if you want to navigate your desktop faster.

Desk setup

  • My laptop is HP Pavilion x360 with Intel Core i5 10th Gen, 8GB RAM, and 512GB SSD storage.
  • The keyboard I currently use is Moonlander Mark 1 with Kailh Box Browns and Lubed Gateron Black Ink switches. It is a split ergonomic and ortholinear mechanical keyboard. Here is my current layout if you are curious, it is Colemak-DH layout with custom macros and layers.
  • Jaybird Vista 2 is my current primary earbuds. Secondary earbuds is Galaxy Buds Pro, I only use it when I can't find my Jaybird.
    I also have Sony WH-1000XM4 as my headset. I don't really use it these days since my ear get sweaty when I wear it.
  • I am sitting at my Staples Hyken Chair. There is a time that I tried working while sitting at floor, but I find it really hard and I can't work for long hours like when I am in chair so I stopped it.
  • My desk is Rakk Malyari. It is a white table with a touch of black accent which make it look pretty nice, it is also really comfortable.
  • If you are working from home, I suggest that you invest for ergonomics. You will thank yourself for it in the future. Your body is your main tool for doing recreations. Invest for it.

Other Gears

  • My french press is Bodum.
  • I'm a simple man, I use Hario Coffee Mill Ceramic Slim to grind my coffee.
  • I use Hario V60 Drip Coffee Scale to measure my coffee and water volume.
